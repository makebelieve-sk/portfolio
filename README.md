# Портфолио

Репозиторий **личного сайта-портфолио** разработчика ([makebelieve-sk](https://github.com/makebelieve-sk)). На сайте — проекты, навыки и сопутствующий контент; всё наполняется **по мере готовности**.

## О проекте

Сайт собирается на **Next.js** (App Router) с **TypeScript** и **Tailwind CSS**. Исходники и публичная часть живут в этом репозитории; смысл репозитория — не шаблон, а рабочая кодовая база портфолио.

## Разделы и темы

**Основные темы и блоки** сайта перечисляются здесь по мере появления страниц и фич. Пока разделы не заведены — позже добавим краткие пункты (название и назначение каждого блока).
