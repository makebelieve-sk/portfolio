# Портфолио

Репозиторий **личного сайта-портфолио** разработчика ([makebelieve-sk](https://github.com/makebelieve-sk)). Это одностраничный лендинг: блоки идут сверху вниз, контент на **русском и английском** (переключатель языка в шапке).

## О проекте

Сайт собирается на **Next.js** (App Router) с **TypeScript** и **Tailwind CSS**. Исходники и публичная часть живут в этом репозитории; это рабочая кодовая база портфолио, а не универсальный шаблон.

## Главы (разделы) сайта

Порядок совпадает с разметкой на главной странице (`src/app/page.tsx`). В шапке **иконка скачивания** по клику скачивает резюме в формате **PDF**.

| Порядок | Раздел                                | Содержание                                                                                                                    |
| ------: | ------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
|       1 | **Вступление** (`Intro`)              | Первый экран: заголовок «Портфолио», имя, роль (Senior Fullstack).                                                            |
|       2 | **Резюме и о себе**                   | Краткое резюме (занятость, формат, локация, опыт), текст о себе, интересные проекты по архитектуре, пет-проекты и npm-пакеты. |
|       3 | **Опыт работы** (`#experience`)       | Компании, периоды, описание задач и стека по местам работы.                                                                   |
|       4 | **Образование** (`#education`)        | Вуз, институт, ступени (бакалавриат, магистратура, аспирантура).                                                              |
|       5 | **Дополнительные курсы** (`#courses`) | Внешние курсы и интенсивы с сертификатами.                                                                                    |
|       6 | **Подвал** (`SiteFooter`)             | Контакты (телефон, почта), QR-коды мессенджеров и профилей (Telegram, VK, GitHub, GitLab, LinkedIn).                          |

Компоненты разделов лежат в `src/components/slides/`, тексты — в `src/locales/<язык>/`.
